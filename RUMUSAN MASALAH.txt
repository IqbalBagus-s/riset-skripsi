1. Identifikasi Masalah
Perkembangan model Convolutional Neural Network (CNN) seperti ResNet-18 telah menunjukkan performa tinggi dalam klasifikasi citra. Namun, model CNN tradisional memiliki keterbatasan dalam menangkap informasi penting secara selektif pada bagian citra yang relevan. Modul attention seperti Convolutional Block Attention Module (CBAM) diperkenalkan untuk mengatasi masalah ini dengan menambahkan mekanisme perhatian spasial dan kanal agar model dapat lebih fokus pada fitur penting.
Meskipun CBAM telah terbukti meningkatkan performa di beberapa dataset besar seperti ImageNet, belum banyak penelitian yang membandingkan secara mendalam kinerja ResNet-18 dan ResNet-18 + CBAM pada dataset Tiny ImageNet, yang memiliki ukuran gambar dan kompleksitas kelas berbeda.

2. Fokus Permasalahan
Masalah utama dalam penelitian ini terletak pada efektivitas integrasi modul CBAM terhadap model dasar ResNet-18 dalam meningkatkan performa klasifikasi citra pada dataset dengan skala menengah seperti Tiny ImageNet. Fokus juga meliputi analisis komparatif performa, efisiensi, dan kompleksitas model.

3. Pertanyaan Penelitian (Research Questions)
Berdasarkan identifikasi dan fokus masalah di atas, pertanyaan penelitian yang dapat dirumuskan adalah:

Bagaimana kinerja model ResNet-18 dalam melakukan klasifikasi citra pada dataset Tiny ImageNet?

Bagaimana pengaruh penambahan modul CBAM terhadap performa model ResNet-18 dalam hal akurasi, loss, dan waktu komputasi?

Apakah model ResNet-18 + CBAM memberikan peningkatan signifikan dibandingkan model ResNet-18 murni pada dataset Tiny ImageNet?

Bagaimana analisis visual (attention map) CBAM membantu model dalam memfokuskan area penting pada citra input?

4. Tujuan Penelitian (Derived Objective)

Mengevaluasi performa model CNN ResNet-18 dalam klasifikasi citra Tiny ImageNet.

Mengimplementasikan dan menguji pengaruh modul CBAM pada arsitektur ResNet-18.

Menganalisis secara kuantitatif dan kualitatif hasil perbandingan antara kedua model.

Memberikan rekomendasi penggunaan modul attention pada model CNN untuk dataset berskala menengah.
